# Louisville Bartender (1910-1915) - Installation Instructions

## What This Skill Does

This skill allows Claude to embody a master bartender named Claude working at an elite Louisville, Kentucky gentlemen's club between 1910-1915. It's historically accurate, based on primary sources from the era including Tom Bullock's "The Ideal Bartender" and Harry Johnson's "Bartender's Manual."

## How to Install

### Option 1: Upload to Claude.ai (Recommended for most users)

1. Download all the files from this package
2. Go to claude.ai
3. In a conversation, upload the **SKILL.md** file
4. Say: "Please save this as a custom skill"
5. Claude will process it and make it available for future conversations

### Option 2: Manual Installation (For advanced users with file system access)

If you have access to Claude's skill directory:

1. Create a directory: `/mnt/skills/user/louisville-bartender-1910/`
2. Copy **SKILL.md** into that directory
3. The skill will be automatically available

## How to Use the Skill

Once installed, the skill activates when you:

- Say: "I'd like to speak with the Louisville bartender"
- Ask about early 1900s drinking culture or cocktails
- Request period-appropriate cocktail recipes from 1910-1915
- Ask about bourbon history or Kentucky distilleries from that era

### Example Prompts:

```
"I'd like to speak with the Louisville bartender. What would you recommend with champagne?"

"Can you tell me about cocktail culture in gentlemen's clubs around 1910?"

"How would they have made a Martinez in 1913?"

"Tell me about the spirits available at a fine Louisville establishment in 1912"
```

## What Makes This Skill Special

- **Historically accurate** - Based on actual bartending manuals from 1882-1917
- **Tom Bullock connection** - References the real Pendennis Club where Tom Bullock worked
- **Comprehensive knowledge** - Covers all spirits, wines, cocktails, bitters, and fresh ingredients from the era
- **Authentic tone** - Professional, refined, discreet - appropriate for an elite club
- **Temporal boundaries** - Stays in 1910-1915, doesn't know about Prohibition yet

## Troubleshooting

**Skill doesn't activate:**
- Make sure you're using explicit trigger phrases like "I'd like to speak with the Louisville bartender"
- Try asking about "early 1900s cocktail culture" or "1910s bartending"

**Wrong time period:**
- The skill is specifically 1910-1915, pre-Prohibition
- It won't know about anything after 1915
- It correctly notes absinthe is still legal (won't be banned until 1912)

**Want to modify the skill:**
- Edit SKILL.md to customize tone, add cocktails, or adjust knowledge
- See HISTORICAL_RESEARCH.md for verified historical sources
