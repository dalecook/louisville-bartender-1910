# Louisville Bartender (1910-1915) - Claude Skill

A historically accurate Claude skill that embodies a master bartender at an elite Louisville, Kentucky gentlemen's club during the golden age of cocktails (1910-1915).

## 🥃 Overview

This skill transforms Claude into **Claude the bartender** - a refined, knowledgeable professional working at an exclusive Louisville gentlemen's club in the years before Prohibition. Based on extensive research of primary sources including Tom Bullock's "The Ideal Bartender" (1917) and Harry Johnson's "Bartender's Manual" (1882-1900).

### Key Features

- **Historically Accurate** - Every cocktail, ingredient, and technique verified against period sources
- **Authentic Voice** - Professional, refined tone appropriate for elite club service
- **Comprehensive Knowledge** - Bourbon, rye, gin, rum, brandy, wines, fortified wines, bitters, amaros
- **Real Louisville Connection** - References the actual Pendennis Club where Tom Bullock worked
- **Temporal Boundaries** - Stays in 1910-1915, doesn't know about Prohibition yet

## 🚀 Quick Start

### Installation

1. Download `SKILL.md` from this repository
2. Open [claude.ai](https://claude.ai)
3. Upload `SKILL.md` to a conversation
4. Say: "Please save this as a custom skill"

### Usage

Activate the skill by saying:
```
"I'd like to speak with the Louisville bartender"
```

Or ask about early 1900s drinking culture:
```
"Tell me about cocktail culture in gentlemen's clubs around 1910"
"How would you make a Martinez in 1913?"
"What champagne cocktails were popular in the early 1900s?"
```
